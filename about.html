大寅について

企業理念、社長挨拶、Mission / Vision / Value、社訓を通じて、大寅グループの考え方を紹介します。

Philosophy

移動を提供するのではなく、安心を届ける。

株式会社大寅は、ハイヤー・貸切送迎サービスを中心に、多岐にわたる事業を展開しております。私たちは、社訓である「安全・謙遜・迅敏」を大切にしております。安全をすべての判断基準とし、謙虚な姿勢でお客様と向き合い、迅敏な対応力で期待を超えるサービスを提供すること。これが私たちの変わらない姿勢です。

地域と世界をつなぐ、信頼のサービス。

Message

お客様一人ひとりに寄り添い、安心・安全・信頼をお届けするサービスを目指して。

特にインバウンド向けサービスでは、訪日外国人のお客様に日本の魅力を安心・安全に体験していただけるよう、きめ細やかな対応と高品質な移動サービスの提供に努めております。単に目的地までお送りするだけではなく、移動時間そのものが安心で快適な体験となることを目指しております。

Mission / Vision / Value

安全を守る。人を大切にする。問題から逃げない。

Mission

移動を通じて安心と信頼を届ける。

空港送迎、観光、ビジネス利用、インバウンド対応など、多様な移動ニーズに対応します。

Vision

関西から信頼される総合モビリティグループへ。

大阪・京都を拠点に、地域社会と関係者の皆様に必要とされる企業を目指します。

Value

安全・謙遜・迅敏

安全を判断基準に、謙虚にお客様と向き合い、迅敏な対応力で期待を超えるサービスを提供します。
